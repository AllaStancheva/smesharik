# Проект: бот-смешарик

## О проекте
Личный Telegram-бот эмоциональной поддержки через юмор. Пользователь пишет своё
состояние («что-то мне грустно», «болит голова», «тяжёлый день»), бот отвечает
связно с репликой — одновременно **поддерживающе и смешно**, смещая фокус в сторону
юмора. Может вплетать любопытные факты, примеры из истории, вдохновляющие истории.

Это личный бот Аллы, не коммерческий продукт.

## Технологии
- Python + aiogram 3.x, режим polling
- LLM через OpenRouter → модель Claude (см. `MODEL` в `llm.py`)
- Весь «характер» бота — в `system_prompt.md`

## Архитектура
- `bot.py` — Telegram-логика: `/start` + обработчик свободного текста → LLM
- `llm.py` — слой OpenRouter (отдельно от бота, чтобы менять модель не трогая `bot.py`)
- `system_prompt.md` — характер смешарика (тон, приёмы юмора, границы)

## Правила работы
- Отвечай и пиши комментарии на русском
- Спрашивай перед изменениями кода
- Не усложняй: бот намеренно простой (без FSM, меню и базы)
- **Перед каждым коммитом** проверяй изменённые файлы на захардкоженные секреты
  (токены, ключи). Секреты — только в `.env` (он в `.gitignore`)
- Системный промпт можно править свободно и тестировать на лету (правка → перезапуск)

## Секреты
- `.env` (не в git): `TELEGRAM_BOT_TOKEN` (от @BotFather), `OPENROUTER_API_KEY`
- Значения вписываются руками, через чат не проводятся
- Резервная копия токена — в Bitwarden

## Запуск локально
```powershell
.\venv\Scripts\Activate.ps1
python bot.py
```
